Comandos básicos para la terminal

Los siguientes comandos permiten moverse a través de la terminal.

Comando Función
pwd Imprimir el working directory.
ls Listado de archivos y carpetas.
cd .. Regresar al directorio anterior.
cd name_directory Cambiar al directorio dado.
clear Limpiar la terminal.
history Historial de los comandos introducidos.
man name_command Mostrar el manual para el comando dado.
cat nombre_archivo.txt Ver contenido de un archivo.

Comandos para modifcar elementos desde la terminal

Los siguientes comandos permiten crear, modificar o eliminar archivos o directorios desde la terminal.

Comando Función
mkdir nombre_directorio Crear un directorio con el nombre dado.
touch nombre_archivo Crear un archivo con el nombre dado.
cp nombre_directorio nombre_nuevo Crear una copia del archivo dado con el nombre dado.
mv nombre_archivo nombre_directorio Mover un archivo a otro directorio, o renombrarlo.
vi nombre_archivo Modificar un archivo.
rm - r nombre_directorio Borrar un directorio.
rm nombre_archivo Borrar un archivo.
