¿Qué es CSS?

CSS, es el acrónimo para Cascading Style Sheets, y es utilizado para determinar la presentación de un documento escrito en HTML o XML. Concretamente, permite definir cómo deben de verse los elementos en la pantalla en términos de diseño, color, fuentes, márgenes, espaciados, entre otros aspectos visuales de una página web.

Elementos que lo conforman

Selectores

Un selector permite apuntar a un elemento o grupo de elementos HTML a los que se les aplicarán estilos definidos. No tiene ninguna restricción sobre qué elementos son seleccionados.

  • Selector de tipo: Permite seleccionar todos los elementos de un tipo específico
  • Selector de clase: Permite seleccionar todos los elementos que tienen una clase específica. Se define utilizando un punto (.) antes del nombre de la clase.
  • Selector de identificador: Selecciona un único elemento que tenga un ID único y específico, determinado con # antes del nombre del ID.
  • Selector universal: Selecciona todos los elementos en una página, determinado con * { } antes de poner los elementos que se definirán.

Diferencias

Selector Caso Sintaxis Ejemplo
Selector por tipo Aplica a todos los elementos del tipo. elemento p { color: red; }
Selector por clase Aplica a elementos con la misma clase. .clase .mi-clase { font-size: 16px; }
Selector por identificador Aplica a un solo elemento único con ese ID. #id #mi-id { background-color: yellow; }

Propiedades en CSS

Son atributos específicos que se quieren modificar o aplicar a un elemento. Las más comunes son:

  • color: Define el color del texto.
  • font-size: Define el tamaño de la fuente.
  • background-color: Define el color de fondo.
  • margin: Define el margen externo de un elemento.
  • padding: Define el espaciado interno de un elemento.
  • border: Define el borde alrededor de un elemento.

Valores

Al trabajar con propiedades, es el valor que se le asigna para definir su comportamiento. Los valores pueden ser diferentes tipos, dependiendo de la propiedad que se está utilizando. Algunos ejemplos de valores son:

  • Colores: red, #ff0000, rgb(255, 0, 0)
  • Tamaños: 12px, 2em, 50%
  • Porcentajes: 50%, 100%
  • Unidades de medida: px, em, rem, %
