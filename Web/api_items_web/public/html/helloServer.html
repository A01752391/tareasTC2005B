Endpoints disponibles

Método Endpoint Descripción
GET / Llama a la página HTML (carga el archivo "helloServer.html")
POST /newItems Agrega uno o más items al catálogo.
GET /items Obtiene todos los items disponibles.
GET /items/1 Obtiene un item por su ID.
PATCH /items/update/1 Actualiza los datos de un item por ID.
DELETE /items/delete/1 Elimina un item por ID.
POST /users/register Registra un nuevo usuario.
GET /users Obtiene todos los usuarios registrados.
GET /users/1 Obtiene un usuario por ID con sus items.
PATCH /users/update/1 Actualiza información del usuario.
DELETE /users/delete/1 Elimina un usuario por ID.
